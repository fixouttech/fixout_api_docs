computeFairnessMetrics(metrics, sFeature, X_test, y_test)

Calculate fariness metrics for a given ensemble of instances.

Supported fariness metrics: Demographic Parity, Equal Opportunity, Predictive Equality.

Check out the enum types : ‘FairMetricEnum’

Parameters:

  • metrics (list) – List of metrics to be calculated.

  • sFeature (SensitiveFeature) – Sensitive feature that will be take into account to calculate fairness metrics.

  • X_test (array-like of shape) – Dataset

  • y_test (array-like of shape) – Predicted outcomes related to X_test.

Returns:

results – List with all requested metrics calculated. Returns an empty list if no metric is informed or a problem when calculating the confusion matrix is found.

Return type:

list of FairMetric

setModelDef(definitions)

Indicates the characteristics of the original model.

    The characteristics of the model, such as: model type, number of expected features, number of expected feature that are non-numeric.
