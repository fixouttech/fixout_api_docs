.. _install:

=======
Install
=======


.. tab:: Using pip

    FixOut is available in **PyPI**.
    
    It is recommended to use a `virtual environment (venv) <https://docs.python.org/3/tutorial/venv.html>`_.
    You need ``python>=3.12``.

    Then, run:

    .. code-block:: bash

        pip install -U fixout

    You can check FixOut's latest version on `PyPI <https://pypi.org/project/fixout/>`_.
