==========
User Guide
==========

.. warning ::

    FixOut is still in development. **The user guide is subject to change.**

FixOut can be used integrated into your ML model's monitoring code or as a separate component. 
In this case, see FixOut Core, to learn which functionalities could be useful.

For monitoring fairness of your ML model via a web interface that generates a dashboard, 
see FixOut Web Dashboard.
